1. 名词解释：
   - 线性时间O(n)：简单查找中需要查找的最多次数为线性时间
   - 对数时间O(log n)：一般指2的对数，即2的多少次方等于n
   - 常量时间O(1)
   - 数组：需要连续占位的一组数据
   - 链表：链表的每个元素都存储了下一个元素的地址，从而使一系列随机的内存地址串在一起，优势在于插入元素，缺点在于读取个别数据时效率过低，因为需要按顺序往下读
